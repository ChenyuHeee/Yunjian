# 云简

> 倬彼云汉，为章于天；执简而书，万舞其韵。

**云简**，一款凝萃东方书写意韵的 Markdown 编辑器。  
化繁为简，以明文心；落键成章，自在从容。

![Icon](docs/icons/yunjian-icon.png)

## 一句话

原生 Swift/SwiftUI 打造的 Markdown 写作应用：编辑、预览、分屏一体，专注写作与发布。

## 亮点功能

- 原生体验：SwiftUI + macOS 菜单栏快捷操作
- 多视图写作：编辑 / 预览 / 边写边看（分屏）
- 文章表现：KaTeX 数学公式、代码高亮
- 图片工作流：支持“粘贴为 PNG”（已保存文档会写入同目录并插入相对路径，便于移动/同步）
- 更顺手的启动与退出：默认主页 + 最近打开；关闭/退出时提示保存未保存内容
- 发布入口：发布菜单内提供“了解 AutoBlog…”快捷跳转（静态博客生成器）

## 最简单的使用方式

1) 下载：打开 Releases 页面，下载最新的 macOS 版本压缩包

https://github.com/ChenyuHeee/Yunjian/releases

2) 安装：解压后将“云简.app”拖入“应用程序”文件夹

3) 打开：首次打开若被系统拦截，可在“系统设置 → 隐私与安全性”中允许，或在 Finder 中右键应用选择“打开”

## 使用提示

- 编辑 / 预览 / 分屏：在“视图”菜单切换
- 贴图：使用“编辑 → 粘贴为 PNG”（已保存文档会把图片写到同目录并插入相对路径）
- 发布：在“发布”菜单里可快速跳转了解 AutoBlog
