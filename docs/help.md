# 云简帮助

本页为 **云简（Yunjian）** 的使用帮助与常见问题说明。

## 快速开始

1. **新建文档**：在菜单栏选择“文件 → 新建”。
2. **打开文档**：选择“文件 → 打开…”，或从“打开最近文件…”中选择。
3. **编辑 / 预览 / 分屏**：在“视图”菜单切换不同写作视图。

## 两种工作方式

云简支持两种使用方式（以适配不同的写作与管理习惯）：

- **资料库模式**：面向日常写作与查找，适合将文稿集中管理。
- **文件夹模式**：面向本地文件夹工作流，适合与 Git、网盘或其他同步工具配合。

你可以在“视图”菜单中切换模式。

## 编辑与写作

- **撤销 / 重做**：使用“编辑”菜单的撤销与重做。
- **查找 / 替换**：使用“编辑 → 查找”相关功能。
- **打字机模式**：在“视图 → 打字机模式”启用后，光标所在行会尽量保持在屏幕中部，适合沉浸式写作。
- **智能排版**：在“编辑”菜单中可启用“在中文和半角的英文、数字之间插入空格”等辅助写作功能。

## Markdown 与语法快捷

云简提供常用 Markdown/写作语法的快捷插入，入口在“语法”菜单。

常见能力包括：
- 加粗、斜体、删除线、链接
- 列表（有序/无序/任务）与引用
- 行内代码与代码块
- 数学公式（行内/块）
- 标题 1～6、分隔线、HTML 注释
- 缩进/反缩进、插入日期/时间

## 图片与附件

- **粘贴为 PNG**：在“编辑 → 粘贴为 PNG”将剪贴板图片以 PNG 形式插入。
- **插入图片或文件**：在“语法”菜单中可插入图片语法或直接选择文件/图片插入。

提示：当文档已保存时，应用会尽量使用相对路径写入图片引用，便于移动与同步（具体行为随当前工作模式与文件位置而定）。

## 发布与导出

在“发布”菜单中可以将内容复制/导出为多种格式（取决于当前文档内容与平台能力）：

- 复制：HTML、富文本
- 导出：Markdown、HTML、PDF、RTF、EPUB、DOCX、TextBundle
- 图片：导出或复制图片

## 常见问题

### 1) 运行后看不到窗口

如果你使用 `swift run YunjianDev` 运行后进程在跑但看不到窗口：
- 在 Dock 里找到 `YunjianDev` 并点一下
- 或使用脚本 `./scripts/run-macos.sh`（会构建 `.app` 并启动）

### 2) “帮助”菜单提示找不到帮助文档

云简已将“帮助”菜单改为指向可用的在线文档与资源入口；如果你仍看到旧的“云简帮助”，通常是打开了旧版本应用或旧构建产物。建议重新构建并启动最新产物。
